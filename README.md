# User Scripts

Collection of various *GreaseMonkey*-compatible user scripts for improving the
modern web experience.

## Scripts

### Google AI-Away(TM)

On *Google* search results and main search page, automatically removes "AI
Mode" buttons, "AI Overviews," and banners promoting Google's vaporware and PR
campaigns.

**Location**: `scripts/google_ai_away.js`

### NYT Cooking to Markdown

On *New York Times Cooking* recipe pages
(`https://cooking.nytimes.com/recipes/*`), exposes a menu button for copying a
Markdown version of the current recipe to the system clipboard.

**Location:** `scripts/nyt_cooking_to_markdown.js`
